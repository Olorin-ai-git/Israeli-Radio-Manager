# Olorin.ai Portal Split (Brand + Content Kit)

This folder implements the **brand/content separation plan**:

- **`olorin.ai`**: non-fraud only (AI agents + vertical solutions like radio station management).
- **`fraud.olorin.ai`**: fraud detection platform + fraud services only.
- **Cross-linking**: only a **single discreet footer link** from `olorin.ai` → `fraud.olorin.ai` (no shared marketing copy).

## Guardrails (non-negotiable)
- No fraud keywords or claims on `olorin.ai` (including SEO metadata, alt text, hidden copy).
- No non-fraud offerings on `fraud.olorin.ai`.
- Separate lead capture + routing (forms, inboxes, CRM pipelines, calendars).
- Distinct visual identity for `fraud.olorin.ai` (can still be “by Olorin.ai LLC” in footer/legal).

## What’s included
- `inventory/`: URL inventory template + audit checklist.
- `olorin-ai/`: information architecture + page-by-page copy for the non-fraud site.
- `fraud-olorin-ai/`: information architecture + page-by-page copy for the fraud subdomain.
- `seo/`: redirect map template + SEO migration checklist.
- `analytics/`: measurement plan + event taxonomy + dashboard spec.
- `implementation/`: rollout runbook + forms/CRM routing spec.
